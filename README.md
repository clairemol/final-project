# final-project
My final 416 project on predicting exoplanet equilibrium temperature using features of the planet and host star. The data come from the NASA Exoplanet Archive and include 211 exoplanets. A baseline was created using Random Forest Regression, and the data was then run through a Random Forest using XGBoost to see which was more accurate. Feature importance was also scored, and the models were re-run with less features as a result of the feature importance scores.
